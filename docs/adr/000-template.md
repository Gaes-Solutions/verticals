## Cómo crear un nuevo ADR

1. Copiar este archivo a `NNN-titulo-corto.md` con el siguiente número libre.
2. Llenar todas las secciones.
3. Estado inicial: **Propuesta**. Pasa a **Aceptada** cuando Gaby la confirma.
4. Si una decisión reemplaza a un ADR previo, dejar el anterior como **Reemplazada por ADR-NNN** y referenciar.
5. Listar el ADR en `STATUS.md` si bloquea trabajo en curso.
